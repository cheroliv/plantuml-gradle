## Quick Rules

- Never commit without explicit user permission
- Never run tests at end of session without being asked
- No secrets or credentials in code or config files
- One file at a time, validate after each change
- Read AGENT.adoc before starting any task

## Context Management — CRITICAL

- **What to do next?** → **BACKLOG.adoc** (authoritative prioritized task/bug tracker)
- **TODO.adoc** → informal ideas memo, NOT a plan, NOT a priority list
- **PROMPT_REPRISE.adoc** → current session mission and next session priorities
