=== 6 Étapes obligatoires — EXÉCUTION COMPLÈTE SANS EXCEPTION

====
**RÈGLE ABSOLUE** : L'agent DOIT exécuter **TOUTES les 6 étapes** **SANS EXCEPTION et SANS OMISSION**.

* ❌ Afficher la checklist de confirmation sans avoir VÉRIFIÉ chaque fichier

== ⚠️ GARDE-FOU — Mise à jour PROMPT_REPRISE.adoc OBLIGATOIRE

=== Problème identifié (Session 90)

**Cause** : Commit effectué SANS mettre à jour `PROMPT_REPRISE.adoc` pour la session suivante.

**Conséquence** : Fichier de reprise obsolète → confusion au chargement suivant.

=== Nouvelle règle absolue (à vérifier AVANT tout commit de fin de session)

[source]
----
✅ CHECKLIST OBLIGATOIRE AVANT COMMIT :

[ ] 1. Archive session N créée dans .agents/sessions/{N}-{titre}.adoc
[ ] 2. PROMPT_REPRISE.adoc mis à jour pour session N+1
[ ] 3. Tags @wip retirés des features complétées
[ ] 4. SESSIONS_HISTORY.adoc mis à jour
[ ] 5. COMPLETED_TASKS_ARCHIVE.adoc mis à jour
[ ] 6. INDEX.adoc mis à jour

⚠️ SI [ ] 1 OU [ ] 2 est coché → INTERDICTION DE COMMITTRE
----

=== Comment appliquer

**En fin de session** (avant `git commit`) :

[source,kotlin]
----
// 1. Vérifier que PROMPT_REPRISE.adoc mentionne "Session N+1"
val promptReprise = readFile("PROMPT_REPRISE.adoc")
if (!promptReprise.contains("Session ${N+1}")) {
    throw IllegalStateException("❌ PROMPT_REPRISE.adoc doit être mis à jour pour Session ${N+1} AVANT commit")
}

// 2. Vérifier que l'archive existe
val archiveExists = fileExists(".agents/sessions/${N}-${titre}.adoc")
if (!archiveExists) {
    throw IllegalStateException("❌ Archive Session ${N} doit être créée AVANT commit")
}

// 3. Vérifier que COMPLETED_TASKS_ARCHIVE.adoc est à jour
val archive = readFile(".agents/archives/COMPLETED_TASKS_ARCHIVE_${YYYY-MM}.adoc")
if (!archive.contains("Session ${N}")) {
    throw IllegalStateException("❌ COMPLETED_TASKS_ARCHIVE.adoc doit contenir Session ${N}")
}

// 4. SEULEMENT APRÈS → git commit autorisé
----

=== Sanction en cas d'oubli

**Si oubli détecté au chargement suivant** :

. ✅ Corriger IMMÉDIATEMENT (créer archive + mettre à jour PROMPT_REPRISE.adoc + COMPLETED_TASKS_ARCHIVE.adoc)
. ✅ Ajouter un commentaire dans le commit suivant : `fix: complete Session N archival`
. ✅ Documenter l'oubli dans `.agents/oubli-archivage-{date}.adoc` pour analyse
