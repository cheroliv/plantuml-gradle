* ✅ `RagConfig` : Paramètre `port: Int = 5432` ajouté

assertTrue(exception.message!!.contains("Connection") || exception.message!!.contains("connect"))

// Tester l'exception encapsulée

    task.reindexRag()

        exception.message?.contains("Failed") == true
    )
}
