== Problèmes rencontrés

=== ❌ Tests trop lents même avec mocks

*Constat* :
* Les tests de performance prennent >5 minutes même avec 10 prompts mockés
* Chaque prompt nécessite un build Gradle complet via TestKit
* Multiplication des builds : 10 prompts = 10 builds Gradle isolés

*Exemple* :
[source,bash]
----
./gradlew cucumberTest --tests "*batch*" # Timeout après 180s
----

*Cause racine* :
* Gradle TestKit crée un projet Gradle isolé par test
* Overhead de démarrage Gradle : ~3-5s par build
* 10 prompts × 5s = 50s minimum, plus exécution réelle

'''
