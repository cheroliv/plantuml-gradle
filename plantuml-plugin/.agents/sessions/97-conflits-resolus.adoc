== Leçons Apprises

=== Pattern à Éviter

❌ **Jamais** dupliquer les steps génériques dans chaque fichier de steps :

[source,kotlin]
----
// MAUVAIS : Dans chaque fichier Steps.kt
@When("I run processPlantumlPrompts task")
fun runProcessPlantumlPromptsTask() { ... }
----

=== Pattern à Suivre

✅ **Centraliser** les steps génériques dans un fichier dédié :

[source,kotlin]
----
// BON : ConfigurationSteps.kt ou CommonSteps.kt
@When("I run processPlantumlPrompts task")
fun runProcessPlantumlPromptsTask() { ... }
----

✅ **Spécialiser** les steps spécifiques au domaine du fichier :

[source,kotlin]
----
// BON : IncrementalProcessingSteps.kt
@When("I run processPlantumlPrompts task again")
fun runProcessPlantumlPromptsTaskAgain() { ... }

// BON : FileEdgeCasesSteps.kt
@Given("a prompt file {string} with UTF-8 content:")
fun createUtf8PromptFile(fileName: String, content: String) { ... }
----
