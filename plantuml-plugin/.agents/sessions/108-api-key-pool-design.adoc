== Contexte

Brainstorming et conception du système de pool de clés API rotatives pour maximiser l'utilisation des quotas freemium multi-comptes.

*Objectifs* :
* Rotation automatique de clés API par provider (Google, HuggingFace, Groq, etc.)
* Maximisation des quotas freemium via multi-comptes
* Gestion des seuils quota (80%) et échecs 429
* Audit logging séparé
* Stockage : GitHub Secrets (JSON consolidé)

'''

== Décisions Architecturales

|===
| # | Décision | Rationale

| 1 | **Stockage** : GitHub Secrets (JSON consolidé) | Plus simple à maintenir, Jackson pour parsing
| 2 | **Sync quota** : Manuel (si API provider le permet) | Pas de scraping, uniquement API officielles
| 3 | **Rotation** : Avant appel + seuil 80% + échec 429 | Proactif + réactif
| 4 | **Poids** : **SUPPRIMÉ** | Abstraction inutile (1 compte = 1 quota)
| 5 | **Fallback** : Oui, avec retry automatique | Continuité de service
| 6 | **Audit log** : Oui, fichier séparé | Traçabilité
| 7 | **MCP** : Non (version ultérieure) | Hors scope Phase 1
| 8 | **Format** : JSON consolidé | 1 secret GitHub = pool complet
| 9 | **HF Spaces** : SpringBoot REST API possible | CPU gratuit, 16GB RAM
| 10 | **1 clé = accès tous modèles** | Simplification (vs par modèle)
|===

'''

== Modèle de Données

----
data class ApiKeyEntry(
    val id: String,
    val email: String,                    // Compte propriétaire

    val provider: Provider,
    val services: Set<ServiceType>,

    val limitValue: Long,
    val consumedValue: Long,
    val thresholdPercent: Int = 80,
    val periodStart: Instant,
    val periodEnd: Instant,
    val resetPolicy: ResetPolicy,
    val lastManualSync: Instant?
)

data class ApiKeyPoolConfig(
    val version: String = "1.0.0",
    val poolName: String,
    val rotationStrategy: RotationStrategy,
    val fallbackEnabled: Boolean,
    val auditEnabled: Boolean,
    val providers: List<ProviderConfig>

----

== Workflow de Rotation

. **Demande de clé** → `ApiKeyPoolManager.selectKey(context)`
. **Vérification quota** → Si ≥ 80% → clé suivante
. **Appel API** → `client.call(selectedKey)`
. **Échec 429** → `markKeyExhausted()` + retry fallback
. **Audit log** → `auditLogger.log(...)`
. **MAJ quota** → `quotaTracker.update(keyId, delta)`
